# Run Tests

Execute the lbxd test suite.

## Tasks

1. Run `cargo test --verbose`
2. Report test results (passed/failed/ignored)
3. For failures, show:
   - Test name and location
   - Expected vs actual values
   - Relevant code context

## After Tests

- Summarize: X passed, Y failed, Z ignored
- If all pass: Confirm test suite is green
- If failures: List each failed test with explanation
