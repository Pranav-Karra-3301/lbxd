## Quick Start

Install lbxd using Homebrew:

```bash
brew tap pranav-karra-3301/lbxd
brew install lbxd
```

Or build from source:

```bash
git clone https://github.com/Pranav-Karra-3301/lbxd.git
cd lbxd
cargo build --release
```
